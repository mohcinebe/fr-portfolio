A propos

Je m'appelle Mohcine et je suis Designer UI/UX et Développeur Web basé au Maroc.

Je suis un grand passionné de nouvelles technologies Web et mobiles, mais mon objectif principal est le développement front-end, tout en approfondissant mes connaissances en back-end, testing et les APIs.

Mes expériences précédentes dans le marketing en ligne et off ligne, combinée avec l'entrepreneuriat, représentent un atout essentiel pour le processus de développement web, car cela m'aide à comprendre les besoins réels de mes clients, en matière d'UI/UX, optimisation pour les moteurs de recherche, fonctionnalités, sécurité et surtout le retour sur investissement.

N'hésitez pas à me contacter pour discuter de votre projet.

Projets personnels

      Outil de tracking pour Instagram

      Un outil de tracking qui permet de suivre des comptes illimités d'Instagram et de générer des rapports d'activité téléchargeable en format PDF, y compris un suivi quotidien, du taux d'engagement, les hashtags viraux du moment, etc.




      Outil d'audit de sites Web

      Un outil pour analyser les sites Web afin de détecter les faiblesses liées à la concurrence et faire des suggestions pour améliorer leurs classements. Il propose également un module d’analyse des frameworks utilisées sur les sites web.




      API de soumission de formulaire auto-hébergée

      Cette API auto-hébergée vous permet de soumettre des formulaires à partir de sites HTML statiques sur le Web. L'API prend vos données de formulaire et les transforme en un email joliment formaté.




      Application de prévision météo

      Une application Web écrite en php et utilisant l’API openweather pour fournir des informations détaillées sur les prévisions météorologiques de n’importe quel endroit.




      Générateur de texte joliment formatté pour les bios des réseaux sociaux

      Petite application générant du texte joliment formatté à utiliser sur des sites Web et des réseaux sociaux tels que Youtube, Twitter, Instagram, etc.




      Vérificateur d'existence de nom d'utilisateur, de produit et de marque

      Vérifiez l'existence d'un nom d'utilisateur, d'un produit ou d'une marque sur de nombreux sites web et les réseaux sociaux populaires.




      Petit lecteur de musique cool

      Un simple lecteur de musique html, CSS et Javascript connecté à Soundcloud.




      TWITCH TV JSON API

      Une application simple utilisant l'API de Twitch, du html/css et du jQuery pour afficher les live en direct et visualiser leurs informations personnelles.
